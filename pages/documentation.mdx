# Documentation

## Get all postal codes
With this easy to use endpoint you will get a list of Postal codes. The request example will return a list of all greek postal codes,street,city and state.

```shell filename="getAllCodes" {3} copy
GET https://api.greek-postal-codes.gr/1/getAllCodes
```
```json filename="result example" 
[
   {
        "street": "ΤΡΙΑ ΜΑΓΑΖΙΑ",
        "tk": 34100,
        "city": "ΧΑΛΚΙΔΑ",
        "state": "ΕΥΒΟΙΑΣ"
    },
    {
        "street": "ΜΕΣΟΓΕΙΩΝ",
        "tk": 15125,
        "city": "ΜΑΡΟΥΣΙ",
        "state": "ΑΤΤΙΚΗΣ"
    },
    {
        "street": "ΤΡΙΤΣΗ ΑΝΤ.",
        "tk": 15235,
        "city": "ΧΑΛΑΝΔΡΙ",
        "state": "ΑΤΤΙΚΗΣ"
    },
]
```
## Get by postal code
You can get results by simply pass as a parameter the postal code you want.

```shell filename="getTk" {3} copy
GET https://api.greek-postal-codes.gr/1/getTk?tk=15125
```
```json filename="result example"
[
    {
        "street": "ΦΛΟΙΑΣ",
        "tk": 15125,
        "city": "ΜΑΡΟΥΣΙ",
        "state": "ΑΤΤΙΚΗΣ"
    },
    {
        "street": "ΑΓΙΟΥ ΑΘΑΝΑΣΙΟΥ",
        "tk": 15125,
        "city": "ΜΑΡΟΥΣΙ",
        "state": "ΑΤΤΙΚΗΣ"
    },
    {
        "street": "ΑΓΙΟΥ ΓΕΩΡΓΙΟΥ (ΣΩΡΟΣ)",
        "tk": 15125,
        "city": "ΜΑΡΟΥΣΙ",
        "state": "ΑΤΤΙΚΗΣ"
    },
]
```
## Get by city
Like we did before on the postal code example you can get results by simply pass as a parameter the city you want.

```shell filename="getCity" {3} copy
GET https://api.greek-postal-codes.gr/1/getCity?city=ΧΑΛΚΙΔΑ
```
```json filename="result example"
[
```

## Get by state
One more simple example for how to get all the data for Attica.You can do the same for any other state. 

```shell filename="getState" {3} copy
GET https://api.greek-postal-codes.gr/1/getState?state=ΑΤΤΙΚΗΣ
```

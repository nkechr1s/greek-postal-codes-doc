# Documentation

## Get all available postal codes

With this easy to use endpoint you will get a list of Postal codes. The request example will return a list of all greek postal codes,street,city and state.

```shell filename="all-postal-codes" {3} copy
GET https://greek-postal-codes.gr/all-postal-codes?page=1
```

```json filename="result example"
{
  "data": [
    {
      "street": "ΤΡΙΑ ΜΑΓΑΖΙΑ",
      "tk": 34100,
      "city": "ΧΑΛΚΙΔΑ",
      "state": "ΕΥΒΟΙΑΣ"
    },
    {
      "street": "ΜΕΣΟΓΕΙΩΝ",
      "tk": 15125,
      "city": "ΜΑΡΟΥΣΙ",
      "state": "ΑΤΤΙΚΗΣ"
    },
    {
      "street": "ΤΡΙΤΣΗ ΑΝΤ.",
      "tk": 15235,
      "city": "ΧΑΛΑΝΔΡΙ",
      "state": "ΑΤΤΙΚΗΣ"
    }
  ],
  "meta": {
    "page": 1
  }
}
```

## Get by postal code

You can get results by simply pass as a parameter the postal code you want.

```shell filename="get-tk" {3} copy
GET https://greek-postal-codes.gr/get-tk/15125
```

```json filename="result example"
{
  "data": [
    {
      "street": "ΦΛΟΙΑΣ",
      "tk": 15125,
      "city": "ΜΑΡΟΥΣΙ",
      "state": "ΑΤΤΙΚΗΣ"
    },
    {
      "street": "ΑΓΙΟΥ ΑΘΑΝΑΣΙΟΥ",
      "tk": 15125,
      "city": "ΜΑΡΟΥΣΙ",
      "state": "ΑΤΤΙΚΗΣ"
    },
    {
      "street": "ΑΓΙΟΥ ΓΕΩΡΓΙΟΥ (ΣΩΡΟΣ)",
      "tk": 15125,
      "city": "ΜΑΡΟΥΣΙ",
      "state": "ΑΤΤΙΚΗΣ"
    }
  ]
}
```

## Get info of a postal code

You can get the city and the state of a postal code by following the next example.

```shell filename="get-info" {3} copy
GET https://greek-postal-codes.gr/get-info/15125
```

```json filename="result example"
{
  "data": [
    {
      "city": "ΜΑΡΟΥΣΙ",
      "state": "ΑΤΤΙΚΗΣ"
    }
  ]
}
```

## Get by city

Like we did before on the postal code example you can get results by simply pass as a parameter the city you want.

```shell filename="get-city" {3} copy
GET https://greek-postal-codes.gr/get-city/ΧΑΛΚΙΔΑ
```

```json filename="result example"

  "data": [

      "street": "ΑΦΡΑΤΙΟΝ",
```

## Get by state

One more simple example for how to get all the data for Attica.You can do the same for any other state.

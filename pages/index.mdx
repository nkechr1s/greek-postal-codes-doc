# Introduction
 
Greek Postal Codes API - Access to Greek Postal Codes Data.

## What is the Greek Postal Codes API?

Lookup postal codes, get all postal codes data that you need for your application and much more with our free postal code API.

## Easy to use API

greek-postal-codes.gr is a tool that allows you to obtain information on different postal codes around Greece. You can get postal codes or get the details of a postal code in general.
Our API is free for use. Checkout the documentation to learn how to use it.

## Get all the postal codes data


You don't need any API key it's very simple to use.

```shell filename="getCity" {3} copy
GET https://beta.greek-postal-codes.gr/1/getCity?city=ΜΑΡΟΥΣΙ
```
